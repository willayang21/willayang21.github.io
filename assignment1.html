Here is all the documentation for assignment 1!
Vgreen = Vred = 1.8V
Vblue = 3.3V
I = 0.02A

For green and red LEDs,
Vgreen + VR = 5V
VR = 5 - Vgreen = 3.2V
R = VR / I = 3.2 / 0.02 = 160Ω
So for resistors in circuits with green or red LED, any resistor that is larger than 160Ω can be used. I chose 1000Ω

For the blue LED,
Vblue + VR = 5V
VR = 5 - Vblue = 1.7V
R = VR / I = 1.7 / 0.02 = 85Ω
So for resistors in circuits with a blue LED, any resistor that is larger than 85Ω can be used. I chose 1000Ω
