I = 5.58W / 12V = 0.465A
So, the maximum current that goes through the transistor should be 0.465A, which is smaller than 0.6A
I also added an ammeter to see how the current changes with the lighting.

For the resistor that connects with the transistor:
The ideal current that flows through the microcontroller is 0.02A,
To protect it, 12V / 0.02A = 600Ω, which means 600Ωcan protect the circuit
in any case with or without the voltage drop.
Here I use 10k just to be safe.

For the resistor with the photoresistor:
5V / 0.02A = 250Ω
So any resistors larger than 250Ω can be used. Here I use 10k to be safe.
